A collection of C++ code intended as a library as well as a playground for practice.
Some of the data structures and reusable code is in .h files


1. Run :
make TEST=filename
Eg: 
$make TEST=binary_tree

2. Debug Using gdb
make DEBUG=filename
Eg:
$make DEBUG=binary_tree
